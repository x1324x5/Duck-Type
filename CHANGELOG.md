# 更新日志 · Changelog

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

### Added 新增
- 🦆 全新 **DuckType（码字鸭）** 品牌与卡通鸭图标（由 `branding.py` 代码绘制，`tools/make_icon.py` 生成 `.ico`）。
- **输入序列浏览与导出**：仪表盘「序列」标签按时间线查看上屏汉字，支持导出 TXT / CSV / JSON。
- **数据管理与隐私**：一键清空、按日期区间删除、`retention_days` 自动保留期限（后台定时清理）。
- **网页设置页**：在仪表盘里直接编辑黑名单、目标、阈值、端口、自启等，免改 `config.json`。
- **自定义时间范围**：除「今天 / 7 天 / 30 天 / 全部」外，可选任意起止日期统计（周报/月报即范围特例）。
- **趋势环比**：关键指标与上一等长周期对比，卡片显示升降幅度。
- **目标 / 连续天数 / 成就**：每日目标进度环、连续打卡 streak、成就徽章墙。
- **趣味榜单**：最爱词、成语/长词、生僻字、只打过一次的字。
- **CLI**：`--clear` 清空数据；`--export` 现在一并导出输入序列 `sequence.txt`。
- 测试套件（`tests/`）+ GitHub Actions 跨平台测试任务。

### Changed 变更
- 数据目录迁移至 `%APPDATA%\DuckType`（数据库 `ducktype.db`、日志 `ducktype.log`）。
- 原生钩子 DLL 更名为 `ducktype_hook.dll`，窗口类/消息名随之更新。

### Fixed 修复
- **单实例保护**：用命名互斥量防止开机自启 + 手动启动产生两个实例互抢钩子与端口。
- **端口占用**：仪表盘端口被占用时自动顺延到邻近可用端口，而不是启动崩溃。
- 移除键盘钩子中未接线的死代码 `on_any`。

## [0.1.0]
