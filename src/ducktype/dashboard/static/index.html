启动与外观

开机自启
登录 Windows 后自动启动 DuckType。
启动即打开仪表盘
启动后直接显示这个统计窗口。
界面主题
可跟随系统，也可以固定亮色或暗色。
语录刷新间隔
每次随机抽取新语录的间隔，单位为秒。
随身鸭热键
全局快捷键，随时打开 / 关闭悬浮的随身鸭。点输入框后按下组合键录制（需含 Ctrl / Alt / Shift / Win）；保存后若提示冲突，说明该组合已被其它程序占用。仅 Windows 有效。
打开
关闭
词库占比·排除后重算
在「词库占比」饼图里点掉某个词后，是否把其余词的百分比按剩余总量重新计算（相加为 100%）。关闭则其余百分比保持相对全体不变。
饼图下载·包含百分比
保存任意占比饼图为图片时，图例里是否标出每一项的具体百分比。关闭则只显示色块、名称与次数。

分析与数据

数据保留天数
填 0 表示永久保留；大于 0 会自动清理更早数据。
分段间隔
超过该停顿或切换程序就视为新的一段，影响分词与序列。
会话间隔
用于计算活跃时长、连续输入和有效删除。
仪表盘端口
仅浏览器预览模式使用；修改后需重启生效。
不统计的程序
每行一个进程名，例如 keepass.exe。
演示数据
用一份示例数据预览仪表盘效果；不会改动你的真实记录，退出即恢复。
改动会自动保存，无需手动点击。
