来交个作业。我用 Vibe Coding 全程「指挥 AI 写代码」，做了一只蹲在 Windows 后台、**只数中文**的码字小鸭——**DuckType（码字鸭）**。

起因特别简单：我想知道自己一周到底码了多少字。结果发现市面上的打字统计工具数的都是「按键次数」，可中文是一串拼音换一个词，**按了多少键跟我真正写了多少字几乎没关系**。能干净地数清「中文上屏量」的本地工具又意外地少。于是干脆自己（指挥 AI）写一个。

![仪表盘总览](screenshots/ducktype-dashboard.png)

## 它到底是个什么东西

一句话：**安静待在系统托盘里，只统计你通过输入法真正敲定的每一个汉字**，再把数据整理成一个好看、能一眼扫过去的本地仪表盘。

- 它**不记你按了什么键**，只看输入法最终上屏的汉字；
- 拼音、英文、数字都不入库，大段粘贴（一次插入超过约 30 字）自动忽略；
- **全程不联网、不上传**，所有记录都躺在你自己选的那个文件夹里；
- 默认跳过密码框，内置常见密码管理器黑名单。

适合长期用中文写东西的人：写论文的研究生、做内容的创作者、爱记笔记的人，以及任何偶尔会好奇「我这周到底码了多少字」的人。

## 打开能看到什么

**一块仪表盘，而不是一张报表。** 首屏先给最关心的几个数字——总字数、平均/峰值速度、活跃时长、修改率——往下是今日目标环、每日产出曲线、按小时的输入节奏，高频字/词、词性分布，还有一张「星期 × 小时」的活跃热力图。点任意一个字或词都能跳进去看它的来龙去脉。

几个我自己最喜欢的设计：

- **报告会替你把数字翻译成人话。** 今日/周/月/年都有一段自然语言小结 + 「行为洞察」卡片（你的高产窗口落在几点、产出比上周期涨没涨、是不是改得有点多），还能一键导出成分享用的方形 PNG 卡片或长图，做周报/年度回顾很顺手。
- **关注词，盯住分词器盯不住的东西。** 人名、项目代号、自己的口头禅，jieba 这类分词器往往切不准，于是从来不进词频榜。加进「关注词」后就直接在上屏序列里逐字精确数：出现多少次、最近一次什么时候、最常在哪个应用里出现。
- **输入序列，是你自己的写作回放。** 真正上屏的中文片段按时间线倒序铺开，可按应用、按关键词筛选，单段复制或整段导出 TXT/CSV/JSON。
- **词库占比。** 内置一份约 3 万条成语词典，用饼图展示这段时间各成语的出现占比；也能上传/粘贴自己的词库。

**随身鸭——写作时只留一个轻量计数器。**

![随身鸭](screenshots/mini.png)

不想开整块看板时，点开「随身鸭」：一个始终置顶的悬浮小窗，用一只无刻度仪表盘的指针实时显示码字速度，配近一分钟的速度曲线、本次会话字数、今日字数与目标进度。全局热键一键开关。

**还有一点点不正经。** 连续打卡、每日目标、成就徽章、最爱词、长词、生僻字、只敲过一次的字……让一个统计工具不至于太严肃。

## 技术上最难的那一关

普通的全局键盘钩子只能看到按键，**拿不到输入法最终提交的汉字**。DuckType 用一个原生的 `WH_GETMESSAGE` 钩子 DLL 注入到 GUI 进程里，直接观察文本提交事件再回传给 Python 主程序：

- 现代程序（微信、VS Code、浏览器、Office、Win11 记事本等）走 **TSF** 文本服务框架；
- 经典 Win32 输入框走 `WM_CHAR` / `WM_IME_CHAR`；
- 两条通路互斥，同一段文本不会被数两遍。

这部分坑最多——光是「同一个字在不同应用里被数两遍 / 候选词框里的拼音被误记」这类问题，就和 AI 反复对了很多轮。这也是 Vibe Coding 体验最神奇的地方：**我能描述清楚「现象」，AI 帮我定位到具体是 TSF 的哪个事件、哪个进程在重复触发**，最后落地成一套去重逻辑。

## 顺便聊聊 Vibe Coding 的体感

整个项目我几乎没手写过几行代码，主要工作是：把想要的体验讲清楚 → 看 AI 实现 → 在真机上跑一遍 → 把 bug 现象描述回去。一些感受：

- **从 0 到能用特别快**，托盘、原生窗口（pywebview + WebView2）、SQLite、图表前端这些「黏合层」AI 写得又快又顺；
- **越底层越要你盯**，像 IME 捕获、PyInstaller 打包后 DLL 环境变量泄漏导致自更新崩溃这种,得靠真机反复验证；
- 它逼着我把「我到底想要什么」想清楚——很多产品决策（只数中文、不联网、密码框黑名单）其实是在和 AI 对话里逐渐定型的。

技术栈：Python + 原生 C 钩子 DLL + pywebview/WebView2 原生窗口 + SQLite + Chart.js，PyInstaller 打包成单个 `DuckType.exe`，MIT 开源。
